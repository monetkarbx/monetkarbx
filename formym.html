Для копирования:

(a + b)² = a² + 2ab + b² - Квадрат суммы

(a - b)² = a² - 2ab + b² - Квадрат разности

(a + b)³ = a³ + 3a²b + 3ab² + b³ - Куб суммы

(a - b)³ = a³ - 3a²b + 3ab² - b³ - Куб разности

a² - b² = (a - b)(a + b) - Разность квадратов

a³ + b³ = (a + b)(a² - ab + b²) - Сумма кубов

a³ - b³ = (a - b)(a² + ab + b²) - Разность кубов
